# 🍕🚀 Pizzafly - Game Design Document 🚀🍕

## 1. 遊戲總覽 (Game Overview)

### 🎯 一句話總結

第一人稱多人 **瘋狂模擬經營遊戲**，玩家不只是在賣披薩，而是在創造屬於自己的「混亂傳奇」：
**甩麵團、搞破壞、耍無厘頭、挑戰物理極限**，在笑聲與崩潰中經營一間不可能存在的披薩店。

### 🌟 核心特色

* 🎨 **無限自由披薩製作**：從香腸到鞋子，從音符起司到恐龍肉，**一切皆可當配料**！
* 🌀 **極限物理互動**：丟麵團、射飛披薩、翻轉重力，甚至把顧客丟進烤箱（？）
* 🤝 **多人瘋狂合作與對戰**：四人合作、兩隊廝殺，不只經營店，還要互相惡搞。
* 📈 **經營＋混亂平衡**：在食材價格波動、奇葩顧客需求與突發事件中，努力求生存。

### 📜 遊戲流程簡述

1. 📲 顧客點餐
2. 👩‍🍳 廚房開始製作（混亂＋物理互動）
3. 📦 裝盒 / 外送 / 亂丟
4. 🧍 顧客吃（或被甩到別的世界）
5. 📊 日結算 + 搞笑報表

## 2. 核心遊戲機制 (Core Gameplay Mechanics)

### 🏪 披薩店經營

* 📱 **智慧手機系統**：顧客評論 🤬🤣、每日新聞 📰、市場行情 📉📈
* 💰 **成本與利潤平衡**：進貨策略、設備維護、員工薪資、裝潢消費
* 🌟 **創意披薩獎勵**：奇怪組合解鎖「稱號」＆吸引特殊顧客 👨‍🍳👽

### 🍳 廚房混亂機制

* 🔄 **重力翻轉器**：整間廚房倒過來 🤸
* 🧲 **磁性披薩盤**：吸走金屬食材 ⚙️（甚至顧客的手錶 ⌚）
* 🌌 **時空傳送門**：披薩瞬移（但可能送到馬桶裡 🚽）
* 🦘 **彈跳地板**：掉落食材彈得滿天飛 🌈

### 🧂 瘋狂配料系統

* 🐟 **活體配料**：魚在披薩上游泳
* 🎭 **情緒配料**：會變色的變色龍肉 🦎
* 🌀 **4D配料**：讓顧客看到未來 ⏳
* 🎶 **聲音配料**：會唱歌的起司 🎵🧀

## 3. 遊戲內容元素 (Game Content Elements)

### 🧑‍🍳 員工與顧客

* **員工**：天才廚師（手滑狂魔）、收納大師、金牌店員。
* **動物員工**：
    * 🐬 **披薩海豚**：智商爆表，但要養在水池
    * 🐒 **快遞猴子**：送餐快，但偷吃
    * 🦜 **收銀鸚鵡**：能記住常客，但會爆料顧客隱私
* **顧客**：奧客、霸王餐客、美食家，還有：
    * ⏳ **時間旅行者**：點恐龍肉披薩 🦖
    * 👽 **外星觀光客**：付奇怪貨幣 🌌
    * 🧍‍♂️ **透明人**：只能靠聲音點單 🗣
    * 🕶 **影子人**：需要調整店燈光照到他 🌙
    * 🤸 **倒立顧客**：要倒立吃 🍕
    * 📸 **實況主**：直播用餐，影響人氣 📹

### 🛠️ 店鋪升級與客製化

* **技能點系統**：得來速 🚗、自動點餐機 🤖、超能力道具
* **店鋪客製化**：裝潢、地板、牆壁，全靠你亂搞
* **日結算系統**：營收報表 + 超搞笑註解（例如：員工偷麵團當枕頭 😴）
* **特殊設備升級**：
    * ⏱ **時間烤箱**：瞬間烤好披薩 🔥
    * 🧠 **心靈感應點餐機**：直接讀顧客想法 🧘
    * 🌐 **全息投影廚房**：透明化流程展示
    * ⚛️ **量子冰箱**：無限儲存，但偶爾吞食材

### 🔒 隱藏要素與彩蛋

* 🌟 **傳說披薩**：召喚神秘顧客
* 🌀 **維度披薩**：給顧客超能力
* ⏳ **時光披薩**：讓顧客回到童年
* 👨‍💻 **開發者披薩**：出現程式碼彩蛋
* 🖊 **玩家簽名披薩**：其他玩家創作隨機入侵
* 🪞 **Meta披薩**：遊戲本身當配料
* 💎 **隱藏彩蛋任務**：
    * 🖊 **玩家簽名披薩**：隨機在世界中出現其他玩家創作的披薩
    * 👨‍💻 **開發者披薩**：用程式碼當配料觸發隱藏彩蛋
    * 🌀 **維度披薩**：製作跨維度配料，顧客獲得暫時超能力

## 4. 任務與事件系統 (Missions & Events System)

### 🌀 日常混亂事件（隨機觸發）

* 🌧 **披薩雨**：天空下披薩，顧客吃到飛天披薩，外送路線完全混亂
* 🥦 **食材叛變**：蔬菜突然有意識，跳開或反擊玩家
* 🔄 **時間循環**：同一個小時重複，顧客會越來越不耐煩
* 🪂 **重力異常**：廚房或餐廳局部失重，顧客和食材漂浮
* 🎭 **情緒配料爆炸**：變色龍肉失控，顏色閃爍到玩家暈眩
* 🛸 **外星訂單**：外星觀光客下訂超稀有配料，付款方式奇怪
* 🐾 **小動物入侵**：快遞猴子或披薩海豚逃出工作區，亂吃食材

### 🎯 迷因級特殊任務（短期挑戰）

| 任務名稱              | 描述                  | 奖勵               |
| ----------------- | ------------------- | ---------------- |
| 🕺 **舞蹈披薩大師**     | 顧客要求邊跳舞邊製作披薩        | 「舞蹈之手」稱號＋特殊舞蹈配料  |
| 🧍‍♂️ **透明人點餐挑戰** | 顧客完全透明，只能靠聲音記點餐     | 「聽覺大師」技能點＋神秘食材   |
| 🦖 **恐龍肉限量訂單**    | 時間旅行者點史前食材          | 超稀有「恐龍肉」配料       |
| 🧟 **殭屍潮外送**      | 顧客全變殭屍，只想要「腦子披薩」    | 「腦汁專家」稱號＋高額小費    |
| 💻 **駭客亂單挑戰**     | 入侵對手訂單，成功混亂三單以上     | 「黑客大師」技能＋特殊道具    |
| 🐸 **青蛙日**        | 食材全變青蛙，還會跳來跳去       | 「青蛙大廚」稱號＋限定配料    |
| ⏳ **未來視披薩**       | 4D配料能讓顧客看到未來，必須精準擺盤 | 「未來之眼」稱號＋特殊技能    |
| 🪞 **Meta披薩挑戰**   | 用遊戲元素當配料，打破第四面牆     | 「開發者之友」稱號＋彩蛋披薩   |
| 🚀 **外星接觸日任務**    | UFO降落，全城點外星披薩       | 超級人氣 + 全店經驗加成    |
| 🎩 **魔法披薩**       | 顧客是魔法師，指定奇幻配料       | 「魔法師助手」稱號＋魔法披薩配方 |

### 🔀 混亂連鎖事件（事件可能互相影響）

* 披薩雨 + 重力異常 = 顧客飛天抓披薩
* 時間循環 + 食材叛變 = 同一塊蔬菜被抓走無限次
* 外星訂單 + 殭屍潮 = UFO 送披薩給殭屍顧客
* 情緒配料 + 透明人 = 顏色閃爍的看不見顧客，完全混亂

### 🏆 任務目標 & 獎勵機制

* ✅ **成功完成事件 / 任務** → 獲得特殊稱號、稀有配料、技能點或超級小費
* 🎁 **累積完成任務** → 解鎖隱藏店鋪裝飾、動物員工、特殊設備

## 5. 多人模式 (Multiplayer Mode)

### 🤝 合作模式
* 四人合作經營，共同面對混亂事件與特殊訂單。

### ⚔️ 競爭模式

* **干擾戰術**:
    * 💻 **駭客攻擊**：竄改對手訂單
    * 🥴 **假食材供應商**：送過期貨
    * 👎 **評論農場**：網軍洗負評
    * 🚧 **交通干擾**：堵住對手門口
    * 🕵️ **竊取廚師**：高薪挖角對方員工
* **聯盟系統**:
    * 🍕 **披薩聯盟**：對抗大型連鎖店
    * 🍅 **配料共享**：互借稀有食材
    * 📡 **情報網路**：分享市場動態
